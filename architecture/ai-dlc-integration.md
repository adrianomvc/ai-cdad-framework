# AI-DLC Integration

## Princípio

> AI-CDAD governa. AWS AI-DLC executa.

AI-DLC entra como execution engine para apoiar:

- Discover
- Design
- Build
- Test
- Deploy
- Observe

AI-CDAD define Entry Lane, Risk Mode, Playbooks, Policies, Gates, Context Budget, Artifact Contracts, Human Approvals e Stop Conditions.
