# PR Review Skill

## Purpose

Review a PR or diff against AI-CDAD constraints.

## When to Use

- Entry Lane: Any
- Risk Modes: FAST/HYBRID/SAFE

## Steps

1. Read runtime-context.md.
2. Read ai-execution-brief.md.
3. Compare diff against scope.
4. Check blocked files.
5. Check tests.
6. Check validation evidence.
7. Check hard stops.
8. Create review notes.

## Stop Conditions

Stop if diff exceeds scope or required approval is missing.
