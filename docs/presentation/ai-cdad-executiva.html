Risk Modes

FAST

Baixo risco, bug local, feature isolada. Governança leve.

HYBRID

Novo + legado, integração existente. Governança moderada.

SAFE

Brownfield crítico, produção, contrato, PII. Governança forte.

Se houver incerteza relevante, o rigor sobe: FAST → HYBRID → SAFE.
