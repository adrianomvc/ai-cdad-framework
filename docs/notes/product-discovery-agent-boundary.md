# Product Discovery Agent Boundary

In the public version of AI-CDAD, the upstream product discovery component is referred to generically as:

```text
Product Discovery Agent
```

It is external to AI-CDAD.

It may represent any product discovery, requirement elicitation or inception workflow used by an organization.

The Product Discovery Agent is responsible for:

- product discovery
- requirement elicitation
- inception
- story creation
- acceptance criteria
- prioritization

AI-CDAD starts after a demand enters the framework.

This public package intentionally avoids organization-specific naming.
